Requerimientos para el Asistente Virtual

Sistema operativo

Este asistente virtual corre bajo virtualmente cualquier sistema operativo. Dentro de los que se destacan: Windows, Linux y MacOS Las versiones mínimas requeridas son:

  • Windows: 8.1, 10 y 11

  • Linux: Ubuntu Ver 18.04, RHEL Ver 6 y 7, CentOS Ver 7

  • MacOS: 10.14 - Mojave

Lenguaje de ejecución necesario

El asistente virtual está totalmente hecho en python. Por esto, es requerido tener python instalado. la versión mínima de python soportada por el asistente es versión 3.9.14 Si no cumple con ese requerimiento, puede actualizar su versión de Python siguiendo las instrucciones en la página oficial de Python: https://www.python.org/downloads/

Una vez satizfechos estos requerimientos, se puede proceder a la instalación.
